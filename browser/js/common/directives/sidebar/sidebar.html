The Rules

The Basics

  • Players: 3 to 5
  • Age: 8+
  • Playing Time: ~20 minutes

The Contents

  • 33 cards numbered 3 to 35
  • 11 coins per person to start

The Idea

This game is a remake of a game called 'No Thanks!' by Thorsten Gimmler. The goal of the game is to have the fewest points. The value of each card is how many points it is worth. Each coin is worth -1 point.

One card will be shown to all players. On each player's turn, they must decide to either:

  • Accept the card; the value of the card will be added to the player's points.
  • Decline the card; one of the player's coins will be added to the card.

The next player faces the same decision: Accept the card (and all the coins on it), or decline it and pay one coin. If a player has no coins on their turn, they are forced to take the card.

Players can see each other's cards, but each player's coins are hidden. The game ends when the cards run out.

The Catch

Unbroken card sequences only count as the lowest number of the sequence. For example, if a player has both the 8 card and the 9 card, those two cards together only count as 8 points. If a player has the cards 17, 18, 19 and 20, all of them together only count as 17 points.

However, at the beginning of the game, 9 random cards will be removed. No player will know which cards are removed.

The Score

Once all of the cards have been taken, each player's total points are calculated by adding up the values of all single cards, plus the value of the lowest card of each unbroken card sequence. For each coin the player has, 1 point is subtracted. The player with the fewest points is the winner.
